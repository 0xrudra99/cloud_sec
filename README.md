# Cloud Security Notes (AWS)

This repository aggregates the cloud security notes I created while learning **AWS security concepts** using a tutor-driven learning approach.

The tutor framework helps me learn topics interactively by introducing concepts, asking questions, and presenting scenarios that require practical reasoning. This forces deeper understanding rather than passive reading.

The goal of this repository is to document my evolving understanding of **AWS security principles, architecture decisions, and real-world scenarios**.

## Learning Method

Instead of traditional notes, these materials were developed through a **question-driven tutor system** that:

* Introduces a cloud security concept
* Challenges understanding with follow-up questions
* Presents practical security scenarios
* Requires explaining decisions and tradeoffs

This method helps reinforce concepts through **active reasoning and repetition**.

## Topics Covered

The repository contains notes and explanations related to AWS security, including:

* IAM (Users, Roles, Policies)
* Resource-based vs Identity-based Policies
* Trust Policies
* AWS STS
* VPC Security Concepts
* Security Groups vs Network ACLs
* KMS and Encryption
* SSM and Automation
* CloudWatch and Monitoring
* Incident Response Concepts

Additional topics will continue to be added as the learning process evolves.
